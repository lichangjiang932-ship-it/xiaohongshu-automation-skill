## 标签模板（12-15个）

### 标签分类
1. **核心标签**（2-3个）：#主题 #关键词
2. **相关标签**（4-6个）：#细分领域 #具体工具
3. **热门标签**（3-4个）：#平台热门 #趋势话题
4. **互动标签**（2-3个）：#分享 #讨论 #推荐

```

设计工具
编程工具
视频剪辑
效率提升

数字生活
EOF
```

### 2. 图片生成
```bash
./scripts/create_images.sh --theme "AI工具" --count 5
```

### 3. 图片上传
```bash
python scripts/upload_images.py --images-dir ./generated_images
```

### 4. 发布准备
```bash
python scripts/publish_post.py \
  --title "🚀 AI工具年度盘点：2024年效率翻倍神器" \
  --content-file content.txt \
  --tags-file tags.txt \
  --generate-script
```

### 5. 执行发布
```bash
./execute_publish.sh
```

## 排版优化指南

### 排版常见问题
1. **内容挤在一起**：缺少换行和段落间距
2. **格式混乱**：原本的结构被破坏
3. **可读性差**：阅读体验不好

### 排版优化技巧
1. **段落间距**：
   - 标题后：2个空行
   - 段落之间：1个空行
   - 分类标题：前后各1个空行
   - 正文与标签：2个空行

2. **换行规则**：
   ```javascript
   // 错误的做法：所有内容在一行
   const badContent = "标题 正文 标签";
   
   // 正确的做法：使用换行符
   const goodContent = `标题\n\n正文\n\n标签`;
   ```

3. **视觉层次**：
   - 使用emoji增强视觉效果
   - 使用符号（•、-、→）创建列表
   - 使用【】或[]标记分类标题
   - 使用数字编号创建有序列表

## 注意事项

### 内容优化
1. **开场吸引**：使用表情符号和强烈语气
2. **结构清晰**：使用分类和编号
3. **价值明确**：突出工具/技巧的价值
4. **互动引导**：鼓励评论和分享
5. **排版美观**：确保段落清晰、间距合适

### 平台规则
1. **字数限制**：标题≤20字，正文≤1000字
2. **图片要求**：3-9张，JPG/PNG，≤32MB
3. **标签数量**：建议12-15个
4. **内容规范**：避免违规内容

### 发布时间
1. **最佳时间**：工作日 12:00-14:00, 18:00-22:00
2. **频率控制**：避免频繁发布
3. **内容规划**：保持内容质量和一致性
